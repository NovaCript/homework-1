Запуск производится из папки blog_api, файлом app.py

POST запрос для создания пользователя:
------------------------------------------------------------------------------------------------
Откройте Postman.
Выберите метод POST.
Введите URL: http://127.0.0.1:5000/users.
Перейдите на вкладку "Body", выберите "raw" и установите тип JSON (application/json).
Введите данные для создания пользователя в формате JSON: { "username": "Новый_пользователь" }.
Нажмите "Send" для выполнения запроса. Проверьте, что пользователь был создан.
GET запрос для получения всех пользователей:
------------------------------------------------------------------------------------------------
Выберите метод GET.
Введите URL: http://127.0.0.1:5000/users.
Нажмите "Send". Убедитесь, что вы получаете список всех пользователей.
POST запрос для создания твита:
------------------------------------------------------------------------------------------------
Выберите метод POST.
Введите URL: http://127.0.0.1:5000/twits.
Во вкладке "Body" выберите "raw" и установите тип JSON (application/json).
Введите данные для создания твита в формате JSON: { "body": "Текст_твита", "author_id": ID_пользователя }.
Нажмите "Send". Проверьте, что твит был создан.
GET запрос для получения всех твитов:
------------------------------------------------------------------------------------------------
Выберите метод GET.
Введите URL: http://127.0.0.1:5000/twits.
Нажмите "Send". Убедитесь, что вы получаете список всех твитов.
PUT запрос для обновления твита:
------------------------------------------------------------------------------------------------
Выберите метод PUT.
Введите URL: http://127.0.0.1:5000/twits/ID_твита.
Во вкладке "Body" выберите "raw" и установите тип JSON (application/json).
Введите данные для обновления твита в формате JSON: { "body": "Новый_текст_твита" }.
Нажмите "Send". Убедитесь, что твит был обновлен.
DELETE запрос для удаления твита:
------------------------------------------------------------------------------------------------
Выберите метод DELETE.
Введите URL: http://127.0.0.1:5000/twits/ID_твита.
Нажмите "Send". Убедитесь, что твит был удален.
Где ID_пользователя и ID_твита - это реальные идентификаторы пользователя и твита соответственно. Убедитесь, что ваш сервер Flask запущен и работает на указанном URL перед отправкой запросов через Postman.
